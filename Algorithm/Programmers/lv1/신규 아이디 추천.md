# 신규 아이디 추천 - 정규식(replaceAll)

##  문제 요약
- 카카오 아이디 규칙에 맞지 않는 아이디를 입력하면 규칙에 맞는 아이디를 추천
- 7단계 처리 과정을 거쳐 유효한 아이디 반환
- 아이디 규칙: 소문자, 숫자, -, _, . 만 사용 / 길이 3~15자 / 마침표는 처음·끝·연속 사용 불가

---

##  왜 정규식(replaceAll)을 사용했는가?

- 특정 패턴의 문자를 찾아서 한번에 제거하거나 치환해야 함
- 반복문으로 하나씩 검사하면 코드가 매우 길어짐
- replaceAll + 정규식을 쓰면 한 줄로 처리 가능

---

##  핵심 정규식 패턴 정리

| 정규식 | 의미 |
|------|------|
| `[^-_.a-z0-9]` | 소문자, 숫자, -, _, . 이 아닌 모든 문자 |
| `[.]{2,}` | 마침표 2개 이상 연속 |
| `^[.]` | 문자열 맨 앞의 마침표 |
| `[.]$` | 문자열 맨 끝의 마침표 |
| `^[.]\|[.]$` | 맨 앞 또는 맨 끝의 마침표 |

---

##  핵심 알고리즘 7단계

### 1단계 - 소문자 변환
```java
new_id = new_id.toLowerCase();
```

### 2단계 - 허용되지 않는 문자 제거
```java
new_id = new_id.replaceAll("[^-_.a-z0-9]", "");
// 소문자, 숫자, -, _, . 을 제외한 모든 문자 제거
```

### 3단계 - 연속 마침표 하나로
```java
new_id = new_id.replaceAll("[.]{2,}", ".");
// 마침표 2개 이상이면 1개로 치환
```

### 4단계 - 처음/끝 마침표 제거
```java
new_id = new_id.replaceAll("^[.]|[.]$", "");
// ^ : 문자열 시작 / $ : 문자열 끝
```

### 5단계 - 빈 문자열이면 "a" 대입
```java
if (new_id.isEmpty()) {
    new_id = "a";
}
```

### 6단계 - 16자 이상이면 15자로 자르기
```java
if (new_id.length() >= 16) {
    new_id = new_id.substring(0, 15);
    new_id = new_id.replaceAll("[.]$", ""); // 자른 후 끝 마침표 제거
}
```

### 7단계 - 2자 이하면 마지막 글자 반복
```java
while (new_id.length() <= 2) {
    new_id += new_id.charAt(new_id.length() - 1);
}
```

---

## 예시 추적 ("...!@BaT#*..y.abcdefghijklm")
```
1단계 소문자 변환     → "...!@bat#*..y.abcdefghijklm"
2단계 특수문자 제거   → "...bat..y.abcdefghijklm"
3단계 연속 마침표     → ".bat.y.abcdefghijklm"
4단계 앞뒤 마침표     → "bat.y.abcdefghijklm"
5단계 빈 문자열 체크  → 변화 없음
6단계 15자로 자르기   → "bat.y.abcdefghi"
7단계 길이 체크       → 변화 없음
→ 최종 결과: "bat.y.abcdefghi"
```

---

## 최종 코드
```java
class Solution {
    public String solution(String new_id) {
        // 1단계: 소문자 변환
        new_id = new_id.toLowerCase();

        // 2단계: 허용되지 않는 문자 제거
        new_id = new_id.replaceAll("[^-_.a-z0-9]", "");

        // 3단계: 연속 마침표 하나로
        new_id = new_id.replaceAll("[.]{2,}", ".");

        // 4단계: 처음/끝 마침표 제거
        new_id = new_id.replaceAll("^[.]|[.]$", "");

        // 5단계: 빈 문자열이면 "a" 대입
        if (new_id.isEmpty()) new_id = "a";

        // 6단계: 16자 이상이면 15자로 자르기
        if (new_id.length() >= 16) {
            new_id = new_id.substring(0, 15);
            new_id = new_id.replaceAll("[.]$", "");
        }

        // 7단계: 2자 이하면 마지막 글자 반복
        while (new_id.length() <= 2) {
            new_id += new_id.charAt(new_id.length() - 1);
        }

        return new_id;
    }
}
```

---

## 느낀 점
--replaceAll() 메서드 자체는 알고 있었지만, 단순히 "ab"를 "a"로 바꾸는 
  수준으로만 사용했었다.
- 이번 문제를 통해 ^ (문자열 시작), $ (문자열 끝), [] (문자 집합), 
  | (또는) 같은 정규식 패턴을 replaceAll() 안에서 활용하는 법을 처음 배웠다.
- 앞으로 문자열에서 특정 패턴을 찾아 제거하거나 치환해야 할 때 
  정규식 패턴을 적극적으로 활용해야겠다.
